    Given $\phi: P\to \RR$ a tropical rational function, the pullback $\phi^*f: Q\to \RR$ is a rational function on $Q$. Furthermore, the pullback function induces a map $\phi^*: \mathrm{Div}(P)\to \mathrm{Div}(Q)$.
