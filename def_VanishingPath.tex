Let $c \in \CritVal(W)$ be a critical point. A \emph{vanishing path} for $c$ is a path $\gamma: [0, \infty) \to \CC$ such that 
\begin{itemize}
\item $\gamma(0) = c$ 
\item $\gamma$ avoids all other critical values 
\item $\gamma(t) = t$ for $t \gg 0$. 
\end{itemize}
